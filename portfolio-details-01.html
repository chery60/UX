DesignMate - A mini design system for your next enterprise product.

Client: Portfolio Project
Category: Design System
Date: 02/November/2022
Tool Used: Figma

DesignMate is a design system that is built focusing on enterprise products. I have tried to incorporate all the basic fundamental elements which can be used to build a simple enterprise product. I have built this design system to help young designers who want be build small enterprise product case studies which they can use to get jobs at product-based companies.

This project is the representation of my learning from a design system course. In this, my fundamental learning was understanding the atomic theory of design. I learned how to apply those fundamentals in my day-to-day design projects. I have used the Figma tool to build this design system.
